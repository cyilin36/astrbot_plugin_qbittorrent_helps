## 0.4.0 - 2026-07-20

### 新增

- 新增任务分类设置和清空功能，仅允许使用 qBittorrent 已有分类。
- 新增任务标签整体替换和清空功能。
- 新增 `/qbt category`、`/qbt tags` 指令及对应 AI tool 操作。
- 搜索结果新增任务当前分类和标签信息。

## 0.3.0 - 2026-07-20

### 新增

- 新增 qBittorrent 任务显示名称重命名功能。
- 新增 `/qbt rename <hash或唯一hash前缀> <新任务名称>` 指令，名称支持空格。
- 统一 `qbittorrent` AI tool 新增 `rename` 操作。

## 0.2.0 - 2026-07-20

### 调整

- qBittorrent 条目搜索改为按添加时间倒序排列，最新添加的条目优先显示。
- 关键词筛选后的结果保持相同的添加时间顺序。

## 0.1.0 - 2026-07-20

### 新增

- 新增 `/qbt` 指令组，支持搜索、磁力链预览、添加和删除下载条目。
- 新增统一的 `qbittorrent` AI tool。
- 支持获取磁力链元数据并按文件编号选择下载内容。
- 支持 qBittorrent Web API 登录、认证重试和最低版本检查。
- 新增 UID 白名单鉴权、预览令牌隔离和删除文件确认保护。
- 新增 AstrBot WebUI 配置 schema 及中英文界面文案。
